# 💊 Pharma_Management_System

## 📌 Project Overview
The **Medicine Inventory & Notification System** is a web-based application developed using **Python (Django)** and **SQLite**.  
It is designed to manage medicine stock, monitor expiry dates, and send timely notifications to ensure efficient inventory handling in medical stores or hospitals.

This system helps reduce manual effort, avoid medicine wastage, and improve operational efficiency.

---

## 🚀 Features

- 🗂️ Manage medicine inventory (Add, Update, Delete)
- ⏳ Track medicine expiry dates
- 🔔 Notification system for low stock & expiring medicines
- 👨‍⚕️ Role-based login system (Admin & Staff)
- 📊 Dashboard for monitoring stock and alerts
- 🔍 Search and filter medicines
- 📁 Organized database management using SQLite
